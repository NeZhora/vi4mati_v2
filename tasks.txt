ЗАДАНИЕ 1
Сдвоенная лабораторная работа 1-2

Решение систем ОДУ

Нежесткая задача:

Практикум часть 1, VIII.11. - один из 5 номеров. (в идеале можно подумать над тем, чтобы система ОДУ задавалась где-то отдельно и можно было бы варьировать размерность и вектор правой части под любую подобную задачу)

Методы, которые надо реализовать:

1. Явные методы Рунге-Кутты 1-4 порядок
2. Явные многошаговые методы 1-4 порядка (по выбору, есть семейство м. Адамса, ФДН или иные смешанные)

Ожидания по работе в команде: 
1. Задание притирочное, в целом каждый член команды может взять на себя одно семейство и реализовать его , а затем вставить свой кусок в единый проект и убедиться, что оно работает - то есть по сути разделить ответственность по методам.

Ожидания по отчету - результат:
1. Построены интегральные кривые y_i(t) (где-то имеет смысл строить фазовые траектории y1(y2) и пр.) решений для разных шагов интегрирования.
2. Суждения, которыми считаете нужным поделиться.

Жесткая задача:

Практикум часть 2, X.9. - один из 14 номеров. (в идеале можно подумать над тем, чтобы система ОДУ задавалась где-то отдельно и можно было бы варьировать размерность и вектор правой части под любую подобную задачу)

Методы, которые надо реализовать (хотя бы 1 из методов должен быть порядка не менее 3): 
0. Посмотреть, что происходит с явными подходами в условиях жесткой задачи.
1. Неявные методы Рунге-Кутта (НРК или ДНРК) не меньше 2 порядка.
2. Методы Розенброка не меньше 2 порядка 
3. Методы Гира
4.* Методы Гира в представлении Нордсика

Ожидания по работе в команде:
1. Покопаться в жесткой задаче и её особенностях. 
2. Выбор того или иного метода из целого семейства может быть как-то обоснован в связи с п.1.
3. Модули для внутренних итераций (решение СНУ, решение СЛАУ)- должны быть интегрированы из предыдущих лабораторных работ прошлого семестра) - это вполне себе внутриинтеграционная отдельная задачка.

Ожидания по отчету:
1. Построены интегральные кривые y_i(t) (где-то имеет смысл строить фазовые траектории y1(y2) и пр.) решений для разных шагов интегрирования.
2. Проведено сравнение с явными реализациями по направлению исследования устойчивости расчета (есть ли плюсы?)
3. Суждения, которыми считаете нужным поделиться.


ЗАДАНИЕ2
Лабораторная работа 3-4. Краевая задача. Задача на собственные значения.

*повышенная сложность

1. Методы решения краевой задачи:

1) метод стрельбы (желательно с учетом расширенной системы, как на семинаре разбирали)
2) метод квазилинеаризации (желательно использовать внутри метод прогонки, т.к. матрица трехдиагональная)
3)* конечно-элементные подходы (метод Бубнова-Галеркина, как на семинаре, или аналоги)

Для отработки методов решения краевой можно решить постановки следующих задач (любые две на выбор):

XI.9.1., XI.9.2., XI.9.3., XI.9.6. или аналогичные.

2. Задача поиска собственных значений: 

1) Методы из п.1. адаптировать под задачу поиска собственных значений.
2)* Метод дополненного (расширенного) вектора (разбирали на семинаре)

Выбрать одну или несколько задач от XI.9.14 до XI.9.21 для численного решения.

 Некоторые напоминания: 
1. В методе стрельбы используются переход к задаче Коши - солверы для ее решения берутся из предыдущей лабы.
2. Метод прогонки - способ решения СЛАУ трехдиагонального вида. Такие матрицы возникнут в методе квазилинеаризации при решении линейной краевой задачи, а также в методе дополненного вектора.

Ожидания по отчету - результат:
1. Построены интегральные кривые y_(x) для соответствующих методов:

1) для метода стрельбы интересно показать, как менялся пристрелочный параметр.
2) для метода квазилинеаризации - количество итераций решения линейной краевой задачи.

2. Для задачи поиска собственных значений выводятся графики интегральных кривых для соответствующих найденных собственных значений. Также полезно будет дать описание алгоритму поиска собственных значений.

3. Суждения, которыми считаете нужным поделиться.
